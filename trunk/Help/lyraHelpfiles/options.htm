Optionen

Konfigurieren Sie lyra nach Ihren eigenen Bedürfnissen!

Anzeige


Sie können hier die Schriftarten für die Standardformatierung und die Spezialformatierung festlegen.
Eine weitere Option bietet die Einstellung für "nicht gesungene" Texte (siehe Übersetzungen).

Der Refrain hat eine besondere Formatierung, die auf der Standardschriftart basiert. Zusätzlich kann die Farbe und die Art der Darstellung ausgewählt werden.
standard, mit "Refrain" markiert: der Refrain wird eingerückt in Standardschriftart und angegebener Farbe dargestellt. Der Refrain-Absatz wird mit Refrain überschrieben.
fett: Der Refrain wird nicht speziell gekennzeichnet. Der gesamte Absatz wird einfach fett dargestellt.


Präsentationsmodus


Wählen Sie hier den passenden Hintergrund für den Präsentationsmodus aus.


Allgemein


  • Info: zeigt eine Info-Datei an (klicken Sie auf den Link). Setzten Sie die Option, wenn Sie die Datei bei jedem Start von lyra anzeigen wollen.
  • rechte Ansicht: vgl. Bildschirmanzeige. Falls markiert, wird die rechte Spalte standardmässig aktiviert.
  • deutscher Titel: aktivieren Sie diese Option, wenn Sie auch bei Übersetzungen den deutschen Titel in der Bildschirmanzeige sehen wollen.
  • Nummer zuerst anzeigen: bewirkt, dass vor dem eigentlichen Lied zuerst während der angegebenen Anzahl Sekunden die Liednummer gross auf dem Bildschirm angezeigt wird. Kann mit der Taste "Ctrl" für manche Funktionen auch umgangen werden, wenn die Option allgemein aktiviert ist (siehe Bildschirmanzeige).


Keine Änderungen zulassen1)

Diese Option kann nur der Administrator (de)aktivieren.
Falls gesetzt, können keine Änderungen übernommen werden (vgl. Speichermanagement).

Option freigeben:


Tippen Sie das Administratorkennwort ein, dann können Sie die Option setzen.


Passwort ändern:


Klicken Sie auf den Link "ändern". Tippen Sie das aktuelle Passwort ins normale Passwortfeld und geben Sie das neue in die beiden anderen Felde ein. Sie können mit "setzen" das neue Passwort übernehmen.


FX-Funktionen


Hier können Sie die Spezialfunktionen (FX-Funktionen) programmieren.
Siehe dazu auch das Kapitel FX-Funktionen.

Klicken Sie auf eine FX-Taste und wählen Sie aus den angezeigten Kategorien die gewünschte aus.
Bestätigen Sie mit 'OK'.
