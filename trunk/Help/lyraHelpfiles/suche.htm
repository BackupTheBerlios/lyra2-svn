Suche

Die Liedtextdatenbank kann selbstverständlich durchsucht werden. Eine Suche ist ja im Grunde selbsterklärend, aber es gibt auch hierzu ein paar nützliche Informationen, die Ihnen die Bedienung erleichtern.



Suchabfrage

Geben Sie hier1) Ihre Suchbegriffe ein!
Es gibt dazu ein paar Kombinationsmöglichkeiten, die je nach Einsatz die Suche recht viel effizienter gestalten als wenn Sie nur nach einem Stichwort suchen.
Beachten Sie: "Lied" und "wir singen ein Lied" werden beide als eine Zeichenfolge behandelt. Ein Lied wird nur angezeigt, wenn darin auch genau diese Zeichenfolge vorkommt. D.h. im zweiten Fall: ein Text, der "ein Lied singen wir" enthält, wird nicht als Resultat angezeigt.

Logische Verknüpfungen:
Sie können Zeichenfolgen zu und/oder-Abfragen verknüpfen.

  • Eine UND-Verknüpfung: "a + b" = a und b müssen enthalten sein.
  • Eine ODER-Verknüpfung: "a - b" = a oder b muss enthalten sein.

Sie können die Verknüpfungen beliebig kombinieren. "und" bindet immer stärker als "oder" (Klammern können keine gesetzt werden).

Die Suche starten Sie mit Klick auf "suchen" oder indem Sie auf "Enter" drücken.

Optionen2)

  • nur Titel: durchsuche nur die Titel. Beschleunigt die Suche je nach Menge gespeicherter Lieder ziemlich stark.
  • gross/klein: wenn aktiviert, wird die Gross-/Kleinschreibung bei der Suche berücksichtigt.
  • ganzes Wort: nur wenn die gesuchten Wörter, resp. Wortabfolgen vollständig übereinstimmen, wird der Liedeintrag als passend angesehen. Z.B. Sie geben "Lied" ein, dann wird "Das Lied ist..." angenommen, nicht aber "Liedtexte".
  • Übersetzungen: aktivieren Sie diese Option, wenn auch die Übersetzungen durchsucht werden sollen. Beachten Sie, dass all die anderen Einstellungen auch für die Übersetzungen gelten.

Resultate

Falls Lieder zu den eingegebenen Kriterien gefunden werden, werden diese in diesem Fenster3) aufgelistet.
Sie können direkt aus diesem Fenster mit Doppelklick einen Liedtext anzeigen.

Quickload4)

Die wohl schnellste Methode, um einen Liedtext anzuzeigen. Sie müssen nur die Liednummer kennen, diese eintippen und dann "anzeigen" klicken (oder "Enter" drücken).
Diese Funktion steht Ihnen in der Bildschirmanzeige auch direkt zur Verfügung.
