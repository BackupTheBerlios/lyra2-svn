Converter

Dieses Konvertierungs-Tool fügt ein Inhaltsverzeichnis (TABLE) aus Titel und Liednummern und eine Auflistung von Texten mit Liednummer (IN) zu einer LTX-Datei zusammen.

Formatieren Sie die Dateien folgendermassen:

TABLE: 1xTabulator, nur einen Titel pro Zeile.
IN: Lieder untereinander aufgelistet. Pro Lied: 1.Zeile: folgende Zeilen:

Speichern Sie die beiden Dateien als TXT-Dateien.

Im Ordner {lyra}\Converter finden Sie das Converter-Tool.
Starten1) Sie es mit Doppelklick auf "Converter.bat". Folgender Text wird in der Konsole angezeigt.

CONVERTER TOOL FOR LYRA
~~~~~~~~~~~~~~~~~~~~~~~

Please indicate the paths of your input-files.
(relative to {converter-dir} or absolute)


The DEFAULT paths of the input-files are:
{converter-dir}\in.txt and {converter-dir}\table.txt
(just press enter to take the default-values)

>> IN File (texts):
...

Geben Sie die Pfade ihrer Input-Dateien an (relativ zum Verzeichnis {lyra}\Converter oder absolut). Oder drücken Sie einfach "Enter", wenn Sie die Standardpfade (IN: {lyra}\Converter\in.txt & TABLE: {lyra}\Converter\table.txt) verwenden wollen (stellen Sie aber sicher, dass Ihre Dateien sich auch dort befinden).

Wenn der Konvertierungsvorgang erfolgreich war, wird "SUCCESSFUL" angezeigt.

Sie finden die Output LTX-Datei im Verzeichnis {lyra}\Converter\out.ltx (der absolute Pfad wird angegeben).
Beachten Sie, dass die LTX-Datei utf-8-formatiert sein muss!

1) dieses Tool ist in Java geschrieben. Das Java Runtime Environment (JRE) muss mindestens in der Version 1.3 installiert sein. Mehr dazu: Sun Java download.
