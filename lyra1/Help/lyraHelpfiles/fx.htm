FX Funktionen

Sie können in den Optionen die Tasten F1-F6 mit speziellen Funktionen (FX-Funktionen) belegen.

  • Geben Sie eine beliebige Datei an. Diese wird aus dem Ansichtsmodus beim Drücken der jeweiligen FX-Taste mit der Windows Standardanwendung für dieses Format geöffnet (z.B. eine Microsoft® PowerPoint® Datei)
  • Oder geben Sie einen ganzen Ordner mit Bildern an. Es werden beim Ausführen dieser Funktion alle Bilder (JPEG,GIF,PNG oder BMP) in diesem Ordner in einem internen Bildbetrachter im Vollbild angezeigt. Vorwärts schalten mit linker Maustaste oder PageUp, rückwärts mit rechter Maustaste oder PageDown. Mit ESC wird die Anzeige geschlossen.
  • Ein mit '-' markierte Funktion bedeutet, dass keine Funktion ausgeführt wird.
