Debugging

Um die Fehlersuche zu erleichtern, verfügt lyra ab der Version 1.2.0d eine debug-Version.
Für Sie augenfälligstes Merkmal ist die debug-Konsole, die im debug-Modus mit lyra zusammen gestartet wird.

Starten Sie lyra im debug-Modus mit dem Kommando "lyra.exe -d" (oder verwenden Sie den Link im Start->lyra->help->lyra debug)


    Die Debug-Konsole kann, wenn Sie sie geschlossen haben, unter Hilfe->"Debug Console" oder mit "Ctrl-D" wieder geöffnet werden.
    Mit der Taste "clear" löschen Sie den ganzen Inhalt der Konsole.
Verwenden Sie diesen Modus, wenn wiederholt System-Fehlermeldungen vorkommen und das Programm aus unerklärlichen Gründen beendet werden muss. Sie können den Informationen in der Konsole eventuell Hinweise über die Fehlerherkunft entnehmen.

Allgemeine Tips:
Überprüfen Sie Ihre Datenfiles und stellen Sie sicher, dass alle Pfade stimmen.
Angaben zu den XML-DTDs finden Sie in der Hilfe (Listen, alle Songs)...

Folgende Registry-Einträge müssen ebenfalls vorhanden sein:
    Windows Registry Editor Version 5.00

    [HKEY_CURRENT_USER\Software\lyra]
    "shnr"="yes"
    "timer"="2"
    "1"="yes"
    "pw"="$$5861637$#"
    "validated"=hex:01
    "url"="data\\curtext.xml"
    "right"="yes"
    "ac"="yes"
    "help"="doc\\lyrahelp.chm"
    "ger"="yes"
    "lists"="data\\lists.xml"
    "schema"="data\\curtext.xsd"

    [HKEY_CURRENT_USER\Software\lyra\fonts]
    "special"="Times New Roman;12;i"
    "color"="00;00;00"
    "refcolor"="0;0;80"
    "transfont"="Microsoft Sans Serif;12;s"
