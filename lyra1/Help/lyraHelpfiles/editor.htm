Editor

Zum Erstellen und Editieren von Liedtexten steht Ihnen ein interner Editor zur Verfügung.



Daten

Falls Sie ein neues Lied erstellen, sollten Sie immer zuerst die Daten eingeben1). Sie können den Text eintippen oder auch aus einem anderen Editor kopieren und einfügen (einfügen mit Ctrl+V).

Liednummer

Sie können jede beliebige Liednummer zwischen 0 und 9999 verwenden. Im Grunde können Sie Liednummern auch doppelt belegen. Bedenken Sie aber dabei, dass die quickload-Funktion nur immer das zuerst hinzugefügte Lied finden und anzeigen wird.
Die Taste "ändern" neben dem Feld "Liednummer" ermöglicht Ihnen, die Liednummer von bereits gespeicherten Liedtexten nachträglich zu ändern. Diese Funktion ist aber mit Vorsicht zu geniessen. Es kann durch die Änderung zu Effizienzverlusten bei der Suche/Quickload-Funktion kommen.


Text formatieren

Links im Fenster finden Sie im Rahmen "Format"2) einige Formatierungsbefehle. Der Text wird nicht direkt im Editor formatiert, sondern mittels sogenanter Tags markiert, die dann in der Bildschirmansicht durch die Formatierung erstetzt werden.
Klicken Sie an die Position, wo Sie die Tags einfügen wollen oder überstreichen Sie den zu formatierenden Text. Dann können Sie die Tags mit Klick auf die jeweilige Taste einfügen.

Tags:

  • Refrain: Text, der mit Refrain markiert ist, wird in der Anzeige im Refrain-Format angezeigt und kann somit besser vom normalen Text unterschieden werden.
  • Spezial: Text, der mit Spezial markiert ist, wird in der Anzeige im Spezial-Format angezeigt. Z.B. die Anzahl Wiederholungen einer Passage kann damit gekennzeichnet werden (z.B. 2x)
    • Diese beiden Formatierungen können in den Optionen konfiguriert werden!

  • Mit Einrücken um 'x' umschlossene Passagen werden um x "nach rechts verschoben" angezeigt.
  • Tab fügt einen Tabulator ein.
  • F formatiert den Text fett, K kursiv.

Der Seitenumbruch break kann nur genau einmal eingefügt werden. Alles, was unterhalb von diesem Tag erscheint, wird in der Ansicht in der rechten Spalte dargestellt! (alles oberhalb in der normalen linken Spalte).


Rückgängig...

Es besteht die Möglichkeit, gewisse Veränderungen rückgängig zu machen. Klicken Sie dafür auf die Taste ganz unten rechts im Format-Rahmen2): ">".
Diese Funktion ist aber nur bedingt einsetzbar. Tag-Editierung kann nur eine rückgängig gemacht werden, direkte Texteingaben im Grunde beliebig viele oder bis zur letzten Tag-Editierung zurück.


Übersetzungen3)

Indem Sie auf die Taste "Übersetzungen >>" klicken, zeigen Sie die Übersetzungen des Liedes in einer eigenen Liste an.
Analog zum Hauptfenster mit der Songliste, erscheinen auch hier Tasten mit den bekannten Befehlen:

  • neu...: öffnet den Übersetzungseditor, wodurch eine neue Übersetzung zum aktuellen Lied hinzugefügt wird.
  • edit: öffnet ebenfalls den Übersetzungseditor. Sie können die Übersetzung analog zum Lied selber auch beliebig editieren.
  • del: löscht die ausgewählte Übersetzung von der Liste.


Mit Klick auf "<< Übersetzungen" blenden Sie die Übersetzungen wieder aus.
Wenn Sie ein Lied neu erstellt haben, müssen Sie unbedingt die Liednummer bevor Sie eine Übersetzung erstellen angeben. (vgl. häufige Fehler).
