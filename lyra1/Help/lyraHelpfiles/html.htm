HTML Seite generieren

Sie können Ihre Liedtexte als HTML-Dateien 2) oder als XML/XSLT-Dateien 3) zur Weiterverwendung mit einem Webbrowser (HTML-Version funktioniert nur mit IEv6+) abspeichern.



Wichtige Angaben1):
  • Projekttitel: Geben Sie Ihrem aktuellen Projekt einen passenden Titel.
  • Basisipfad: Geben Sie den Pfad an, wohin die Dateien gespeichert werde sollen.
  • Die restlichen Angaben werden aus den obigen zwei generiert. Ändern Sie nur etwas daran, wenn Sie ganz sicher sind, dass Sie damit nichts überschreiben, was nicht überschrieben werden sollte.

Wählen Sie darunter den Modus aus.
2) generiert reine HTML-Dateien. Die Liedtexte werden in eigenen HTML-Dateien abgespeichert und über eine Linkseite verknüpft. Der Browser muss Frames unterstützen.
3) XML/XSLT-Version - unterstützt eine sehr moderne Technologie, wobei mitels sogenanntem Stylesheet (XSLT) die HTML-Seite direkt im Browser aus einer XML-Datei generiert wird.

Die generierten Dateien entsprechen den Spezifikationen des W3Cs (XHTML1.0/CSS).

Wichtig!

Als Quelle für die HTML- resp. XML/XSLT-Generierung wird jeweils die Liste des aktven Registers im Benutzerfenster verwendet!
Wenn Sie z.B. eine HTML-Seite von einer persönlichen Liste erstellen möchten, müssen Sie das Register "persönliche Liste" aktivieren, bevor Sie über das Menü den Punkt "HTML Seite generieren" anwählen.
