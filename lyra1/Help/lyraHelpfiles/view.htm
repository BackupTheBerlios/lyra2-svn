View (Vollbild-Ansicht)

Die Hauptfunktion von lyra ist neben dem Verwalten von Liedtexten in einer Datenbank die effiziente Generierung von Vollbildansichten aus den Daten eines Liedes.



Eine View gliedert sich in Liednummer/Titel, die gesondert angezeigt werden1), und den Liedtext.
Liednummer und Titel können nicht speziell formatiert werden, sie dienen eigentlich nur der Identifizierung des geladenen Textes. Den Hauptteil macht der Liedtext aus2). Der im Editor formatierte Text wird hier gemäss den in den Optionen eingestellten Eigenschaften geladen und dargestellt. Der Text kann in der View nicht editiert werden.

Wichtig und interessant sind vor allem die Kontrollbefehle3), die direkt von der View aus ausgeführt werden können.

Sie können das vorhergehende, resp. das nachfolgende Lied des aktuellen Liedes direkt aus der View laden. Klicken Sie dafür auf den Menüknopf3)->"Navigation"->"vorwärts"/"rückwärts" oder drücken Sie "PageUp"/"PageDown". Der Abfolge der Lieder entspricht jener der Liste, woraus Sie das aktuelle Lied geladen haben! Z.B. die Songliste, die Suchresultat-Liste oder eine eigene Liste.
Wenn Sie die Option "Liednummer beim Start anzeigen" aktiviert haben, können Sie die Anzeige dennoch umgehen, indem Sie "Ctrl"-"PageUp"/"PageDown" drücken.

Der Liedtext wird entweder als ein einziger Block oder in zwei Spalten angezeigt. Falls Sie zwei Spalten verwenden wollen, müssen Sie sicherstellen, dass die Option "rechtes Fenster benützen" gesetzt ist (Standardeinstellung kann in den Optionen festgelegt werden). Damit das Programm weiss, wie es den Text aufteilen soll, müssen Sie im Editor einen Seitenumbruch (break-tag) einfügen.

Falls Sie zu einem Lied Übersetzungen gespeichert haben, erscheint dazu ein Menüeintrag "Übersetzungen" mit den jeweiligen Untereinträgen. Klicken Sie auf einen Eintrag um eine Übersetzung zu laden. Wenn Sie die rechte Spalte verwenden, wird die Übersetzung in der rechten Spalte angezeigt, links bleibt der Originaltext. Ansonsten wird die Übersetzung wie ein normales Lied angezeigt. In den Optionen können Sie dazu festlegen, ob der deutsche Titel ebenfalls angezeigt werden soll. Über den Menüpunkt "Originaltext anzeigen" können Sie die Übersetzung wieder ausblenden.

Schliessen Sie die View über den Menüeintrag "schliessen" oder einfacher indem Sie Esc oder Alt+F4 drücken.

Quickload

Analog zur Quickload-Funktion Hauptfenster "Suche", können die Lieder auch direkt in der View über ihre Liednummer angesprochen und geladen werden. Geben Sie einfach die Liednummer im Feld unter dem Menüknopf3) ein und drücken Sie "Enter" (Wenn Sie die Option "Liednummer beim Start anzeigen" aktiviert haben, können Sie die Anzeige dennoch umgehen, indem Sie "Ctrl"-"Enter" drücken.)
Um das Eingabefeld aus der View zu aktivieren, drücken Sie die Taste "F11"!
